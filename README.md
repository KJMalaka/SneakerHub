Sneaker Hub SA

A modern, responsive e-commerce website for Sneaker Hub SA, South Africa’s sneaker destination.

Overview

Originally built in 2023 as a school project, the site has been redesigned with improved UI, smoother user experience, and modern web standards.
Sneaker Hub SA showcases premium sneakers from leading brands including Nike, Converse, New Balance, Puma, and Vans.

Focus areas: authenticity, quality, and customer satisfaction.

Features

Responsive design for desktop, tablet, and mobile

Product catalog with detailed sneaker listings

Shopping cart with local storage

Contact form for support and inquiries

Order receipt page for completed purchases

About page with company story and team profiles

Clean UI with gradients, shadows, and animations

Pages

index.html – Home with hero and featured products

products.html – Product catalog with filtering

cart.html – Shopping cart management

about.html – Company info and team

contact.html – Contact details, FAQ, and form

receipt.html – Sample receipt for orders

Tech Stack

HTML5 – Semantic structure

CSS3 – Flexbox, Grid, animations

Google Fonts – Custom typography

Setup

Download or clone the repository

Open index.html in a browser

Explore pages via the navigation menu

This is a static website. No server setup required.
